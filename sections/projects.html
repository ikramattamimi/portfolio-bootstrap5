Trashclean

Description

Website that aims to manage waste collection activities in a village. This website is designed to provide two different usage schemes. The first scheme is for users who do not need to log in, who can access the company profile, blog, information and tutorial views. The second scheme is for users who require authentication, including citizen, admin, and super admin users. The content in this website including home, about, catalog, contact, and blog is customable and can be managed by the super admin.

Salary Payment

Description

A website whose main objective is to facilitate the process of sending salary slips via email. This application allows users to send pay slips to a number of employees easily, by uploading an excel file containing the data needed to make pay slips.
