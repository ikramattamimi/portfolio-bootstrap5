Trashclean

Description

Website that aims to manage waste collection activities in a village. This website is designed to provide two different usage schemes. The second scheme is for users who require authentication, including citizen, admin, and super admin users. The content in this website including home, about, catalog, contact, and blog is customable and can be managed by the super admin.
